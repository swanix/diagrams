🎨 Guía del Tema Custom Compacto

💡 Concepto: El tema custom compacto incluye solo las variables CSS esenciales, haciendo el archivo mucho más pequeño y fácil de mantener.

📋 Ventajas de la Versión Compacta

  • Archivo más pequeño: Solo ~50 líneas vs ~300 líneas
  • Fácil mantenimiento: Menos código duplicado
  • Mejor rendimiento: CSS más eficiente
  • Actualizaciones automáticas: Usa el CSS principal
  • Menos errores: Menos código que mantener

🚀 Cómo Usar

1. Abrir el Tema Custom Compacto

Abre el archivo index-custom-compact.html en tu navegador:

http://localhost:8000/index-custom-compact.html

2. Personalizar el Tema

Edita solo las variables CSS en el archivo HTML:

📝 Estructura del Tema Custom Compacto

<style> .theme-custom { /* Colores Base */ --bg-color: #f6f7f9; --text-color: #222; --node-fill: #fff; --label-border: #bdbdbd; --link-color: #999; --node-stroke-focus: #1976d2; /* Panel Lateral */ --side-panel-bg: #fff; --side-panel-text: #222; --side-panel-header-bg: #f8f9fa; --side-panel-header-border: #dee2e6; --side-panel-border: #e0e0e0; --side-panel-label: #666; --side-panel-value: #222; /* Selección de Nodos */ --node-selection-border: #e0e0e0; --node-selection-focus: #1976d2; --node-selection-hover: #bdbdbd; --node-selection-selected: #1976d2; /* Imágenes y Filtros */ --image-filter: grayscale(30%); /* Loading */ --loading-color: #1976d2; --loading-bg: #ffffff; /* Fondo */ --bg-image: none; --bg-opacity: 1; /* Mensajes */ --error-message-color: #999999; } </style>

🎯 Variables Incluidas

--bg-color
Color de fondo principal
--text-color
Color del texto principal
--node-fill
Fondo de los nodos
--label-border
Borde de los nodos
--link-color
Color de las conexiones
--node-stroke-focus
Color de enfoque de nodos
--side-panel-bg
Fondo del panel lateral
--side-panel-text
Texto del panel lateral
--node-selection-focus
Color de selección de nodos
--image-filter
Filtro para thumbnails
--loading-color
Color del spinner
--bg-image
Imagen de fondo (opcional)
--bg-opacity
Opacidad del fondo

📊 Comparación: Compacto vs Completo

Aspecto Versión Compacta Versión Completa
Tamaño del archivo ~50 líneas ~300 líneas
Mantenimiento Fácil Complejo
Actualizaciones Automáticas Manuales
Rendimiento Mejor Normal
Flexibilidad Alta Máxima

⚠️ Consideraciones

Nota: La versión compacta:
  • Depende del archivo sw-diagrams.css principal
  • Se actualiza automáticamente con mejoras del CSS principal
  • Es más fácil de mantener y menos propensa a errores
  • Recomendada para la mayoría de casos de uso
