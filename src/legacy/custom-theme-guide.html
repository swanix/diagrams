🎨 Guía del Tema Custom

💡 Concepto: El tema custom es un tema integrado directamente en el HTML que se puede editar fácilmente y se usa por defecto.

📋 Características

  • Integrado en HTML: Todo el CSS está en el archivo HTML
  • Por defecto: Se carga automáticamente al abrir la página
  • Personalizable: Fácil de editar directamente en el código
  • Persistente: Se guarda en localStorage
  • Compatible: Funciona con todos los temas existentes

🚀 Cómo Usar

1. Abrir el Tema Custom

Abre el archivo index-custom.html en tu navegador:

http://localhost:8000/index-custom.html

2. Personalizar el Tema

Edita directamente las variables CSS en el archivo HTML:

<style> .theme-custom { /* Cambia estos valores para personalizar */ --bg-color: #f6f7f9; /* Color de fondo */ --text-color: #222; /* Color de texto */ --node-fill: #fff; /* Fondo de nodos */ --link-color: #999; /* Color de enlaces */ --node-selection-focus: #1976d2; /* Color de selección */ } </style>

🎯 Variables Principales

--bg-color
Color de fondo principal de la página
--text-color
Color del texto principal
--node-fill
Color de fondo de los nodos
--label-border
Color del borde de los nodos
--link-color
Color de las líneas de conexión
--node-selection-focus
Color cuando se selecciona un nodo
--side-panel-bg
Fondo del panel lateral
--side-panel-text
Texto del panel lateral
--loading-color
Color del spinner de carga
--bg-image
URL de imagen de fondo (opcional)
--bg-opacity
Opacidad del color de fondo (0-1)

🎨 Ejemplos de Personalización

Ejemplo 1: Tema Azul Corporativo

.theme-custom { --bg-color: #f5f7fa; --text-color: #2c3e50; --node-fill: #ffffff; --label-border: #3498db; --link-color: #3498db; --node-selection-focus: #2980b9; --side-panel-bg: #ffffff; --side-panel-text: #2c3e50; --loading-color: #3498db; }

Ejemplo 2: Tema Verde Naturaleza

.theme-custom { --bg-color: #f0f8f0; --text-color: #2d5016; --node-fill: #ffffff; --label-border: #4caf50; --link-color: #66bb6a; --node-selection-focus: #388e3c; --side-panel-bg: #ffffff; --side-panel-text: #2d5016; --loading-color: #4caf50; }

Ejemplo 3: Tema con Imagen de Fondo

.theme-custom { --bg-color: #1a1a1a; --text-color: #ffffff; --node-fill: rgba(255, 255, 255, 0.9); --label-border: #ffffff; --link-color: #ffffff; --node-selection-focus: #00ff88; --bg-image: url('https://images.unsplash.com/photo-1632059368252-be6d65abc4e2?w=1920&q=80'); --bg-opacity: 0.3; --loading-color: #00ff88; }

⚠️ Consideraciones Importantes

Nota: Al editar el tema custom, asegúrate de:
  • Mantener la estructura CSS correcta
  • Usar valores de color válidos (hex, rgb, rgba)
  • No eliminar las variables necesarias
  • Probar en diferentes navegadores

🔄 Cambiar entre Temas

Puedes cambiar entre el tema custom y otros temas usando el selector en la esquina superior derecha:

  • Custom: Tu tema personalizado
  • Light: Tema claro por defecto
  • Dark: Tema oscuro
  • Vintage: Tema retro
  • Pastel: Tema suave
  • Cyberpunk: Tema futurista
  • Neon: Tema neón

💾 Persistencia

El tema seleccionado se guarda automáticamente en el localStorage del navegador, por lo que se mantendrá la próxima vez que abras la página.
